Datasets & Tools


This page is under construction.

AccessLens Dataset is a dataset augmented using ADE20K, of indoor scene images to automate precise inaccessibility detection.

LumiPrint Gallery introduces applications and Grasshopper scripts to customize 3D printable visual capabilities using phosphorescent materials.

Multti-tach is a tool to modify 3D model for better interlocking structures in multi-material 3D printing in FDM.

3D4ALL is a dataset of 3D models in public repository for content moderation.
